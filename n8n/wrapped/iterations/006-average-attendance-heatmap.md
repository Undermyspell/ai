# Iteration 006: Durchschnittliche Anwesenheitsquote pro Monat Heat Map

# Ziel 
Es soll eine zweite Heatmap enstehen, die die durchschnittliche Anwesenheitsrate pro Stammtisch anzeigt
Beispiel: An einem Donnerstag waren alle da -> 100%, am nächsten nur die Hälfte -> 50% , durchschnittle Anwesenheit 75%

# Vorgehen
- Die neue Heatmap kann genauso wie die andere Heatmap aussehen und auch die gleichen Animation haben
- Es soll nicht die durchschnittle Anwesenheit pro User sonder pro Stammtisch Tag angezeigt werden.
